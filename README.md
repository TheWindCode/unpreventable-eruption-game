# unpreventable-eruption-game

"Unpreventable Eruption" is a fast-paced 3D stategy game, made in 48 hours for Ludum Dare 50 Compo. The theme was "Delay the Inveitable".

It was made in Unity 2022.2.12f1, using URP.
